AS.P.2

Name

Develop AgileSafe Practices Compliance Argument

Actor

User / Method

Description 

Creating or editing an AgileSafe Practices Compliance Argument for a given standard.

Steps

1.     Prepare a structure for Practices Compliance Argument based on a chosen standard and AgileSafe Practices Compliance Pattern.

2.     Fill the structure with practices from AgileSafe Practices Knowledge Base using the AgileSafe Practices Compliance Template

3.     [Optional] Analyse and assess the ‘strength’ of the Argument

Artefacts

AgileSafe Practices Compliance Argument

 

 

AS.P.3

Name

Adapt AgileSafe Practices Compliance Argument

Actor

User

Description 

Customizing the appropriate AgileSafe Practices Compliance Argument to the needs of a given project. This process should be carried out for each standard the project is expected to conform with.

Steps

1.     Select the appropriate Practices Compliance Argument, depending on the standard you want to conform with.

2.     Narrow down the list of practices in the Practices Compliance Case to those selected for the Project Practices Set.

3.     Based on PROJECT COMPLIANCE TEMPLATE PATTERN and the edited Practices Compliance Case [from step 2] prepare the AgileSafe Project Compliance Template

Artefacts

AgileSafe Project Compliance Template
