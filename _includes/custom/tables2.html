AgileSafe Practices Knowledge Base

A knowledge base of software development and management practices, both agile and disciplined, created for the needs of projects that use AgileSafe.

 

AS.A.3

Name

AgileSafe Practices Compliance Template

Type

Supplied by the method

Description

A single template for developing AgileSafe Practices Compliance Arguments.

 

AS.A.4

Name

AgileSafe Practices Compliance Argument

Type

Supplied by the method / created with a support of the method

Description

An argument that is developed based on AgileSafe Practices Compliance Template, a given standard and applicable practices from AgileSafe Practices Knowledge Base.

 

AS.A.5

Name

AgileSafe Project Compliance Template

Type

Created with a support of the method

Description

A template for developing AgileSafe Project Compliance Argument, a tailored version of AgileSafe Practices Compliance Argument containing only practices relevant to the given project.

 

AS.A.6

Name

AgileSafe Project Compliance Argument

Type

Created with a support of the method

Description

The main argument to be used for monitoring conformance with a given standard. Contains the evidence produced in the process of applying practices presented in the Project Compliance Template for the given project.

 

AS.A.7

Name

Project characteristics

Type

External

Description

An analysis of the project characteristics, formal or informal, which contains necessary knowledge to decide on the most suitable software development practices for the project. A guide on the information that should be provided in such analysis is presented in the 5.1 Project characteristics
